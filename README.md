# Anime Downloader


Desenvolvido para rodar em GNU/Linux


>Dependecias:<i>
- python 2.7
- wget
- requests
- sys (pylib)
- os  (pylib)</i>

-----------
>Para inciar:

- Copie o link da pagina inicial de algum anime do site: https://animesonline.cc

- Execute o programa: <code>python animedown.py</code>
